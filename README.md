# SpringMockServiceApplication
Веб-приложение для проверки деплоя в minikube 

## Доступные ендпоинты

 - `GET` `/actuator/health` - проверка работы приложения
 - `GET` `/actuator/prometheus` - возвращает метрики для Prometheus
 - `GET` `/api/sentiment` - ендпоинт имитирует анализ текста, для переданного текста в параметре `text` возвращает случайный результат `позитивный`, `негативный` или `нейтральный`
